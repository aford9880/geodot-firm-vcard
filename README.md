# ГеоТочка — лендинг геодезической компании

Одностраничный сайт с интерактивной Leaflet-картой как центральным элементом:
зоны обслуживания, выполненные объекты, калькулятор стоимости, форма заявки
с отправкой в Telegram и на email. Дизайн в топо-картографической тёмной тематике
с акцентами геодезиста (оранжевый) и геодезической зелени.

## ✨ Что внутри

- **Hero = карта** — полноэкранная Leaflet-карта с зонами обслуживания (GeoJSON-
  полигоны), плавающий оверлей с заголовком и CTA, живая карточка «бригада на выезде»,
  поиск точки по координатам или адресу (через Nominatim OSM) с авто-определением
  зоны (`src/js/map.js`).
- **Зоны обслуживания** — отдельная интерактивная карта с переключателями зон,
  расцветкой по тарифам, всплывающими попапами с тарифом и временем реакции
  (`src/js/zones.js` + `src/data/zones.geojson`).
- **Выполненные объекты** — карточки с процедурно генерируемыми «топо-фрагментами»,
  фильтры по типу работ, модальное окно с мини-картой объекта на Leaflet
  (`src/js/objects.js` + `src/data/objects.json`).
- **Калькулятор стоимости** — выбор типа работ, площади/количества, срочности и
  зоны → интерактивный расчёт диапазона цены с breakdown по компонентам
  (`src/js/calculator.js`).
- **Live-журнал работ** — бегущая строка последних выполненных задач; ротация
  карточки «бригада на выезде» (`src/js/ticker.js`).
- **Процесс работы** — горизонтальный таймлайн от заявки до акта.
- **FAQ** — аккордеон с поиском (`src/js/faq.js`).
- **Форма заявки** — имя, телефон, тип работ, адрес объекта, площадь → отправка
  в Telegram (Bot API) + email (mock/Formspree) (`src/js/form.js`).
- **Полный SEO**: `robots.txt`, `sitemap.xml`, Open Graph/Twitter, JSON-LD
  `LocalBusiness`, semantic HTML.
- **Аналитика** — mock gtag/dataLayer, трекинг кликов и событий (`src/js/analytics.js`).
- **Интерактивности**: кастомный курсор-crosshair, магнитные кнопки, 3D-наклон
  карточек, scroll-анимации (IntersectionObserver), прелоадер с журналом загрузки
  в стиле терминала геодезиста.
- **Адаптивность** — бургер-меню, брейкпоинты в SCSS, корректное
  `invalidateSize()` карт при ресайзе и попадании в видимую область.

## 🛠 Технологии

| Категория   | Технологии                          |
|-------------|-------------------------------------|
| Сборка      | Vite 5                              |
| Стили       | SCSS (один `main.scss`, без libs)   |
| Скрипты     | Vanilla JS (ES-модули)              |
| Карты       | Leaflet 1.9 + CARTO Dark Matter tiles |
| Данные      | GeoJSON зон, JSON объектов          |
| Формы       | Telegram Bot API + Email mock       |
| Шрифты      | Space Grotesk, Inter, JetBrains Mono |

Рантайм-зависимость — только `leaflet`. Остальное dev-зависимости.

### Секции лендинга (`index.html`)

2. `#ticker` — live-журнал работ
3. `#services` — услуги
4. `#objects` — выполненные объекты + модал-карта
5. `#zones` — интерактивная карта зон
6. `#calculator` — калькулятор стоимости

9. `#faq` — частые вопросы

## 🚀 Установка и запуск

Требуется Node.js 18+.

```bash
npm install
cp .env.example .env     # при необходимости пропишите токены (см. ниже)
npm run dev              # dev-сервер
npm run build            # сборка в dist/
npm run preview          # предпросмотр сборки
```

## ⚙️ Конфигурация (.env)

Все переменные опциональны — без них сайт работает на заглушках/mock.

```env
# Telegram: куда падают заявки с формы
VITE_TELEGRAM_BOT_TOKEN=
VITE_TELEGRAM_CHAT_ID=

# Email: endpoint сервиса-relay (например Formspree/Web3Forms). Без значения — mock в console.
VITE_EMAIL_ENDPOINT=
```

### Telegram

1. Создайте бота через [@BotFather](https://t.me/BotFather) → токен.
2. Узнайте `chat_id` через [@userinfobot](https://t.me/userinfobot) или ID группы.
3. Подставьте в `.env` (или прямо в `src/js/form.js`).

Пока значения плейсхолдеры — форма имитирует успешную отправку.

|            | Значение             |
|------------|----------------------|
| Фон        | `#0b1418`            |
| Surface    | `#0f1c20`            |

| Акцент 2   | `#2dd4a7` (geode)    |

Фон страницы — топо-контуры на SVG, рисуются прямо в CSS.

## Главная аудитория → лиды

- Карта задерживает внимание и даёт ощущение мгновенной пользы (зона и тариф
  по клику, поиск по адресу).
- Калькулятор даёт прозрачность по цене и отводит страх «скрытых» счетов.
- Карточка «бригада на выезде» и live-журнал работ создают ощущение активности
  и реальности компании.
- Все CTA ведут в одну точку: форму заявки.
